11am - 12:00pm

You the Real MVP!

Chelsea Carter

Join Chelsea as she navigates the world of MVP development tools — Adalo versus FlutterFlow to determine the real MVP. Chelsea, a backend software engineer, shares her firsthand experiences and insights gained from comparing these platforms. Through practical examples and friendly comparisons, attendees will discover which tool reigns supreme for their projects. Whether you're a seasoned pro or new to MVP development, Chelsea's session offers a fun and informative guide to selecting the best software for your needs. Get ready to crown the real MVP!

How to make a Rest API in Node.js

Marcus Westbrooks

I'm going to demonstrate how to build a basic REST API with standard crud functions. We will download Node.js and express, then from there we will build a class for posts. This class will allow us to make and store posts, edit them, and delete them. We will then interact and test this API using Postman.

Using SwiftCharts to Visualize your Data

Nicholas Gordon

Using the SwiftUI Framework of "SwiftCharts" where we make a simple expense tracking app. The presentation will cover what is SwiftCharts and how to incorporate into SwiftUI. Mainly showing the Bargraph and how we can take user input to show the data visually. Can also show how to change the color of the graph and even the pie chart and line graph supplied by the SwiftCharts framework.

5pm - 6:00pm

DevOps: From Startup's to Scale

Dwayne Toler

Startup's aren't leveraging the power of DevOps effectively. Startup's can use DevOps to automate the deployment of their application to cost effective cloud platforms to start and then move to tier 1 clouds such as AWS, Azure and GCP as they gain traction. They can do this without having to change their complete workflow.
